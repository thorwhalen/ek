ek.tools

Small, CLI-friendly functions over the ek core (the dispatch surface).

These are thin, string-in/value-out wrappers designed for triple dispatch (CLI via argh, and later HTTP/UI). The single _dispatch_funcs list is the SSOT of what ek.__main__ exposes, so there is no duplicated command registration.

Run them from the shell:

python -m ek cer "hello wrld" "hello world"
python -m ek where
python -m ek check tesseract
